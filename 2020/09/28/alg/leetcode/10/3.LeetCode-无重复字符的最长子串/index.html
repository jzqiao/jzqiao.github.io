LeetCode-无重复字符的最长子串

发表于 2020-09-28 分类于 算法 Valine：

给定一个字符串，请你找出其中不含有重复字符的最长子串的长度。

  • 输入: “abcabcbb”
    输出: 3
    解释: 因为无重复字符的最长子串是 “abc”，所以其长度为 3。

  • 输入: “bbbbb”
    输出: 1
    解释: 因为无重复字符的最长子串是 “b”，所以其长度为 1。

  • 输入: “pwwkew”
    输出: 3
    解释: 因为无重复字符的最长子串是 “wke”，所以其长度为 3。
